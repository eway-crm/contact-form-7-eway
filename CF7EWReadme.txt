== Description ==

This plugin provides ability to create eway lead when user sends email 
through contact form 7 Wordpress plugin.

== Installation ==

1. Copy this plugin to wordpress plugin directory.
2. Log into the administration page via your eWay-CRM Web Service address and credentials
3. Map custom fields in "Mapping" section if you wish so.
4. Enjoy your Contact Form.
